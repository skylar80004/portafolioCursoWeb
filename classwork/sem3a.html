Semana 3: Sintaxis HTML5 y CSS

15 elementos básicos importantes de HTML:

  1. Header de texto:< h1 > < h2 > < h3 > < h4 >

  2. Parrafo: < p >

  3. Horizontal ruler: < hr >

  4. Link o dirección: < a >

  5. Lista ordenada: < ol >

  6. Lista desordenada: < ul >

  7. Elemento de lista: < li >

  8. Imagen: < img >

  9. Separador: < div >

  10. Text span: < span >

  11. < style >: Etiqueta de estilo usada para usar CSS en la misma línea

  12. < head >: Metadatos acerca del documento incluyendo enlaces o definiciones de scripts y hojas de estilos

  13. < script >: Define ya sea un script interno o un enlace hacia un script externo

  14. < body >: Representa el contenido principal de un documento HTML

  15. < cite >: Representa el título de una obra

15 elementos de HTML5:

  1. < aside > : Define contenido relacionanado tangentemente al contenido que lo rodea.

  2. < figcaption > : Define un subtítulo o "caption" para un elemento < figure >

  3. < header > : Suele contener las secciones de cabecera

  4. < footer > : Se usa para contenido ubicado al fondo de la página web

  5. < main > : Delimita el contenido principal de la página web web

  6. < mark > : Tiene como objetivo llamar la atención del lector hacia un texto importante. web

  7. < nav > : Denota un sección con enlaces de navegación. web

  8. < section > : A diferencia de div, el elemento section demarca un agrupamiento temático de contenido. Cada sección tiene tipicamente un elemento encabezador y demás contenido asociados a otros divs, párrafos,etc. web

  9. < details > : Produce un caja expandible para mostrar información adicional. web

  10. < summary > : Elemento opcional que resume los contenidos del elemento details del padre. web

  11. < datalist > : Define opciones predefinidas para el control de inputs. Trabaja de forma similar a un autocompletado. web

  12. < keygen > : Define un campo generador de llave-pares para formularios. web

  13. < progress > : Define una barra de progreso. web

  14. < output > : Define el resultado de un cálculo. web

  15. < audio > : Define sonido o contenido musical. web

20 propiedades más utlizadas en CSS3:

  1. background: Define una variedad de propiedades de fondo dentro de una sola declaración

  2. border: Define el ancho, estilo y color de todos los cuatro lados del borde de un elemento

  3. color: Especifica el color del texto de un elemento

  4. opacity: Especifica la transparencia de un elemento

  5. height: Especifica la altura de un elemento

  6. width: Especifica la anchura de un elemento

  7. content: Inserta contenido generado

  8. font: Define una varidad de propiedades de la fuente de letra dentro de una declaración

  9. justify-content: Especifica como items flex se alinean a lo largo del eje principal del contenedor flex después de que cualquier todos los tamaños y auto margenes se hayan resuelto.

  10. flex:Especifica los componentes de un tamaño flexible.

  11. margin: Establece el margen de todos los cuatro lados de un elemento

  12. column-count: Especifica el número de columnas en un elemento con múltiples columnas

  13. outline: Establece la anchura, estilo y color de los cuatro lados de la línea de afuera de un elemento

  14. padding: Define el padding de los cuatro lados de un elemento

  15. text-align: Define el alineamiento horizontal del contenido de una línea.

  16. text-align: Especifica el método de justificación a usar cuando la propiedade de "text-align" está establecida como "justify"

  17. text-shadow: Añade una o más sombras al texto de un elemento

  18. vertical-align: Establece el posicionamiento vertical de un elemento relativamente respecto a la línea base del elemento.

  19. position: Especifa cómo está posicionado un elemento

  20. display: Especfica como un elemento se muestra en pantalla
